Description:

Bibliosync is an online library website that helps users view and search for books to borrow.
The website has two types of users: admins and regular users where each user have a unique username and email.
Also the website contains a navigation bar which differs in the options whether you're an user or an admin.


Admin Features:

  • Sign up (fill in form “username, password, confirm password, email, and is_admin”).
  • Login
  • Add new books (ID, book name, author, category, description).
  • View list of available books.
  • Select a book and edit its details.    **Not Functional Yet**
  • Delete a book.

User Features:

  • Sign up (fill in form “username, password, confirm password, email, and is_admin”).
  • Login.
  • Search for books by title, author, or category.
  • View list of available books marked either available or not available.
  • Select a book and view its details.
  • Borrow a book (only applicable if the book is available).
  • View list of borrowed books.
  • Request adding a book.
